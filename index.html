Two Primary Data Sources

1. AskRAND Dataset (Full Text)

2. RAND API (Publication Metadata)

  • What: Official RAND publication database with hand-curated metadata
  • Content: Authors, titles, abstracts, publication details, research topics
  • Created by: RAND Publications staff for each official publication
  • Purpose: Provides rich bibliographic and subject metadata
  • 2020+ Coverage: 5,085 total publications

Key Distinction: AskRAND has full document text but limited metadata. RAND API has rich metadata but no full text content.
